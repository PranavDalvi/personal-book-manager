# 📚 Book Manager (Next.js + MERN Stack)

A full-stack **Book Management application** built with **Next.js, Node.js APIs, MongoDB, and React**.
Users can **register, login, and manage their personal reading list** with tags, status tracking, and filters.

---

# 🛠 Tech Stack

### Frontend

- Next.js (App Router)
- React
- Tailwind CSS
- shadcn/ui components
- react-select
- Sonner

### Backend

- Next.js API Routes
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs for password hashing

---

# ⚙️ Setup Instructions

1. Download / Clone the repository

2. Install dependencies

```
npm install
```

---

3. Configure environment variables

Create a `.env` file and copy the contains of .env.sample

---

4. Run the development server

```
npm run dev
```

App will start at:

```
http://localhost:3000
```
